Ce git est pour le projet EMI d'arthur DEROISSART, Samuel BESLON et Paul BUET.

Il a pour but de modéliser la sédimentation d'abords dans un modèle 1D puis dans un modèle 3D (mésoNH).

Il est composé des scripts comp_sedimentation.py qui compare deux sédimentations

# Projet EMI 2026 Phy-nh : "Vers une revisite de la sédimentation dans Arome"

## Installations et configurations nécessaires

pour se mettre sur l'environnement : python -m venv mon_env ; source mon_env/bin/activate ; pip install -r requirements.txt
pour lancer les scripts : python3 Scripts/entree.py
